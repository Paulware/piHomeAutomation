Device List

When you create a new sd card, all the devices that had been previously added to your system will be forgotten
So before you wipe out your old sd card, run a php program named:
actionsSnapshot.php
To capture the prevous devices (on the old system):
  • Power up your old pi with the old devices
  • Connect to its wifi accesspoint if it has it
  • From the browser execute the url/command:
    http://192.168.4.1/Paulware/actionsSnapshot.php
  • This will create the .php file: restoreSnapshot.php
  • Using winscp, copy this file to your new sd card, boot directory
  • Power up the new sd card, in a raspberry pi
  • Using winscp copy the restoreSnapshot.php from the boot directory to the /var/www/html/Paulware directory
  • Using a browser, run restoreShapshot.php by opening the url:
    http://192.168.4.1/Paulware/restoreSnapshot.php
    Your devices should now reappear
Note: if your old pi does not have a wifi access point available you will need to change 192.168.4.1 to whatever ip address the pi is located on your lan.
